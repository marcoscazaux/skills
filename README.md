## Available Skills

| Nombre     | Descripción                                                                                                                                                                                                                                                                                                            | Install URL                                                                                               |
| ---------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------- |
| angular    | Best practices for Angular (v20+) development. Covers TypeScript 5.9 strict mode, standalone components, signal-based inputs/outputs, `inject()` DI, native control flow (`@if`/`@for`/`@switch`), `host` property, `OnPush` change detection, Reactive Forms, lazy loading, SCSS with BEM, and WCAG AA accessibility. | [![skills.sh](https://skills.sh/b/marcoscazaux/skills)](https://skills.sh/marcoscazaux/skills/angular)    |
| bem        | Complete reference guide for the BEM (Block Element Modifier) CSS/SCSS naming methodology. Covers blocks, elements (`__`), modifiers (`--`), mix technique, `.is-*` state helpers, flat BEM nesting, specificity, and naming variants.                                                                                 | [![skills.sh](https://skills.sh/b/marcoscazaux/skills)](https://skills.sh/marcoscazaux/skills/bem)        |
| bemit      | Combined BEM + ITCSS system. Adds namespace prefixes (`.o-`, `.c-`, `.u-`, `.t-`, `.s-`, `.js-`, `.qa-`, `._`), responsive `@breakpoint` suffixes on utilities, themes, scopes, hacks, and extended file structure. Requires `bem` and `itcss` skills.                                                                 | [![skills.sh](https://skills.sh/b/marcoscazaux/skills)](https://skills.sh/marcoscazaux/skills/bemit)      |
| commitlint | Guides AI agents to write compliant git commit messages using commitlint. Covers config detection, reading enforced rules via `--print-config`, writing Conventional Commits format, validation before committing, and self-correction from hook rejections (no `--no-verify`).                                        | [![skills.sh](https://skills.sh/b/marcoscazaux/skills)](https://skills.sh/marcoscazaux/skills/commitlint) |
| itcss      | Complete guide to ITCSS (Inverted Triangle CSS/SCSS) by Harry Roberts. Covers all 7 layers (Settings, Tools, Generic, Elements, Objects, Components, Utilities), file structure, `@use` import order, compatibility with Angular, BEM, Tailwind, CSS `@layer`, and design tokens.                                      | [![skills.sh](https://skills.sh/b/marcoscazaux/skills)](https://skills.sh/marcoscazaux/skills/itcss)      |
